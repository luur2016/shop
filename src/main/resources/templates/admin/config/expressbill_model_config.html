操作提示
  • 平台现有运单模板列表
  • 点击设计按钮可以对运单模板布局进行设计，点击测试按钮可以对模板进行打印测试，点击编辑按钮可以对模板参数进行调整
  • 设计完成后在编辑中修改模板状态为启用后，商家就可以绑定该模板进行运单打印
  • 点击删除按钮可以删除现有模板，删除后所有使用该模板的商家将自动解除绑定，请慎重操作
