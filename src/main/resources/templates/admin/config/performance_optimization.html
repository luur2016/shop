性能优化

  • 性能优化
操作提示
  • 启用内存优化功能将会大幅度提升程序性能和服务器的负载能力，内存优化功能需要服务器系统以及PHP扩展模块支持
  • 目前支持的内存优化接口有Redis、Memcache、eAccelerator、Alternative PHP Cache(APC)和Xcache，系统将会依据当前服务器环境选用接口
  • 内存接口的主要设置位于 config.ini.php 当中，您可以使用EditPlus（禁止使用记事本、写字板）通过编辑 config.ini.php 进行高级设置
  • 内存优化功能设置的内容，只有开启内存缓存后才会生效，更改缓存详细参数需编辑 config.ini.php 文件
